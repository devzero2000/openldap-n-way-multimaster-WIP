### OpenLdap N-Way Multimaster WIP

 Questo repository contiene toy scripts, playbooks ansible e documentazione a
corredo per configurare il prodotto OSS OPENLDAP come un 3-Way replication
Multimaster.  Questi semilavorati sono usati per creare un lab di sviluppo.
Naturalmente nulla di quanto qui presente è adatto o puo' essere usato per
ambienti di produzione di qualunque natura.
